# Security Policy

## Supported Version

当前接受安全修复的版本为 V1.5 最新发布版本。

## Reporting a Vulnerability

请优先使用 GitHub 仓库的 **Security > Report a vulnerability** 私密报告功能。报告中请包含：

- 受影响的版本与部署环境
- 可复现步骤与预期影响
- 必要的请求或响应片段，敏感值请脱敏
- 你已经尝试过的缓解措施

请不要在公开 Issue 中提交数据库文件、`config.php`、管理员账号、密码、Cookie、Token、服务器地址或完整日志。

维护者确认问题后会评估影响范围、准备修复并协调披露时间。尚未发布修复前，请避免公开可直接利用的细节。
